# DapStore - Lab Phase Draft

## Project Overview
**DapStore** is a MERN stack e-commerce website draft. Users can browse products, manage profiles, and interact with a basic shopping experience. This Lab Phase draft demonstrates the initial setup of both frontend and backend, ready for further development.

---

## Features Implemented (Draft)
- **Backend**
  - Express server running on port 5000
  - Dummy API route (`GET /`) returning a welcome message
  - Placeholder User model (`models/User.js`)
  - Placeholder authentication routes (`routes/auth.js`) for signup and login

- **Frontend**
  - React app running on port 3000
  - Pages: Home and Profile
  - React Router setup for navigation between pages

---

## Planned Features
- User authentication (signup/login)
- Product search and details
- Shopping cart
- Order history
- Integration of frontend and backend API
